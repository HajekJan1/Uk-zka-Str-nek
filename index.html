O mně

Ahoj! Jsem 15letý web developer a začínám se specializovat na vývoj moderních a responzivních webových stránek. Moje cesta do světa programování začala před pár lety a od té doby jsem se naučil Javu, CSS a HTML. Mým cílem je vytvářet weby, které jsou nejen funkční, ale i vizuálně atraktivní a přívětivé pro uživatele. Věřím, že každý web by měl mít svou vlastní jedinečnou atmosféru a zážitek, který si návštěvníci užijí. Rád se neustále učím nové technologie a zlepšuji své dovednosti, abych mohl poskytovat kvalitní služby. Těším se na možnost spolupracovat na tvých projektech!

Moje dovednosti

1

Ovládám HTML, CSS na úrovni semi-seniora. Javascript na úrovni začítečníka.

2

Dokážu perfektně komunikovat, stránku dodat na Vámi zvolený termín.

3

Nabízím i služby spojené s Microsoft Office a několika jinými programovacími jazyky na základní úrovni.

4

Jsem ambiciozni, ochotný se učit. Do své práce dám vše. Nabízím stránky za bezkonkurenční ceny.
